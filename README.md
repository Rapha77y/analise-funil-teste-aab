# analise-funil-teste-aab
ANALISTA DE DADOS 
🧪 Novo projeto no portfólio: Análise de Funil de Vendas + Teste A/A/B!

Trabalho de análise comportamental para um app de produtos alimentícios, com mais de 244 mil eventos registrados por 7.551 usuários.

📊 O que foi analisado:

🔹 Funil de vendas — onde os usuários abandonam o app?
• Maior gargalo: 38% dos usuários somem logo na tela inicial antes de ver as ofertas
• Quem chega ao carrinho quase sempre compra — apenas 5,2% de abandono
• Taxa de conversão total: 47,7% do início ao pagamento

🔹 Experimento A/A/B — a nova fonte de texto intimida os usuários?
• 3 grupos testados (~2.500 usuários cada)
• Teste A/A validou que os grupos eram equivalentes ✅
• Resultado: nenhuma diferença estatisticamente significativa entre fonte nova e antiga
• Conclusão: mudança de fonte pode ser implementada com segurança 🟢

🛠️ Ferramentas utilizadas:
Python · Pandas · Plotly · SciPy · Statsmodels · Jupyter Notebook
